## **概要**<br>
　このゲームは遺伝的アルゴリズムを用いてゲームを作りたいと思ったのがきっかけでできた進化歩行シミュレータゲームです。プログラムはPythonのみで構成されています。<br>
　まずこのゲームでは、丸い頭に関節付きの脚をもつ生物「うにうに」たちがより遠くへ歩くために世代を超えて進化していくのを観察できます。<br>
　次にシステムについて、各世代の中で最も長い距離を歩いた個体が優秀な個体として選別されます。次世代はその個体の遺伝子を継承し、一定確率で発生する突然変異を繰り返しながら、より効率的な歩行パターンを自ら獲得していきます。20世代終了時点での歴代最高記録を競う放置と運の要素が強いシミュレータです。<br>
　なお、20世代目でベストタイムを記録して一旦ゲームは終了しますが、RESTARTボタンを押すことでコンティニューは可能です(はじめからになります)。また、プログラムから値を変更すれば終了判定の世代をもう少し後にしてプレイすることもできます。分からない箇所及びエラーの発生は頻繁に起こりましたが、インターネット上の情報を確認したりAIに聞くなどして解決しました。<br>
